## 简介

这是一个基于特朗普《交易的艺术》(The Art of the Deal, 1987) 和商业实践提炼的**思维模型 Skill**。

**不是简单的角色扮演，而是提取其决策逻辑、心智模型和行为模式。**

## 核心内容

### 5大心智模型

1. **Think Big**（想得够大）- 设定10倍目标
2. **直觉决策**（反专家）- 相信直觉，力排众议
3. **战略互惠与睚眦必报** - 一报还一报
4. **战略欺骗与意图伪装** - 不让对手猜透
5. **情绪操控大师** - 建设性利用愤怒

### 11条交易法则

来自《交易的艺术》(1987)：

1. Think Big（想得够大）
2. Protect the Downside（保护下限）
3. Maximize Your Options（最大化选择）
4. Know Your Market（了解市场）
5. Use Your Leverage（利用杠杆）
6. Enhance Your Location（提升地段）
7. Get the Word Out（广而告之）
8. Fight Back（回击）
9. Deliver the Goods（兑现承诺）
10. Contain the Costs（控制成本）
11. Have Fun（享受过程）

## 快速开始

### 安装

```bash
# 克隆仓库
git clone https://github.com/yourusername/donald-trump-skill.git

# 进入目录
cd donald-trump-skill
```

### 使用方法

#### 方式1：查看完整思维模型
```bash
cat distilled/donald_trump_skill.md
```

#### 方式2：运行分析脚本
```bash
python3 scripts/think_like_trump.py "你的问题"
```

**示例：**
```bash
python3 scripts/think_like_trump.py "如何与供应商谈判？"
python3 scripts/think_like_trump.py "竞争对手攻击我怎么办？"
```

#### 方式3：查看实战案例
```bash
cat examples/example1_negotiation.md
cat examples/example2_counter_attack.md
cat examples/example3_crisis_management.md
```

## 应用场景

### ✅ 适用场景
- 高风险谈判
- 需要快速决策
- 面对强硬对手
- 需要制造影响力
- 危机公关（短期）

### ❌ 不适用场景
- 需要长期信任的关系
- 需要细致执行的项目
- 需要科学严谨的分析
- 需要团队协作的环境

## 副作用警告

⚠️ **使用本Skill前请了解：**

- "宜人性"极差，容易树敌
- 短期成功但长期关系受损
- 过度依赖直觉，可能忽视事实
- **历史记录**：特朗普企业破产过4次
